# 全局快捷键
## 使用方法
对于大多数[动作](#动作系统)，您可以通过修改`{userDir}/copytranslator/shortcut.json`来绑定全局快捷键。
::: warning
当快捷键已被其他应用程序占用时，绑定将失败。
:::
每条记录的形式为：`action:accelerator`。

## 默认快捷键定义
```json
{
"listenClipboard": "CommandOrControl+J"
}
```
